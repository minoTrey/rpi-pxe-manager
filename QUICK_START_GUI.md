# 🚀 빠른 시작 가이드 - GUI 버전

## 1️⃣ 실행하기

### 가장 간단한 방법
```bash
cd rpi-pxe-manager
./run_gui.sh
```

### 또는
```bash
python3 pxe_gui.py
```

## 2️⃣ 첫 화면

프로그램을 시작하면 다음과 같이 구성됩니다:

## 3️⃣ 클라이언트 추가하기

1. **🖥️ 클라이언트 관리** 클릭
2. **➕ 추가** 버튼 클릭
3. 정보 입력:
   - **시리얼**: `12345678` (8자리)
   - **MAC**: `e3:0f` (마지막 4자리만!)
   - **IP**: `192.168.0.100` (자동 제안됨)
4. **추가** 버튼 클릭

### 💡 MAC 주소 자동완성 마법!
- `e3:0f` 입력 → 자동으로 `88:a2:9e:1b:e3:0f` 완성
- `e30f` 입력해도 OK!

## 4️⃣ 서비스 관리하기

1. **🚀 서비스 관리** 클릭
2. 각 서비스마다 버튼 제공:
   - **▶️ 시작**: 서비스 시작
   - **⏸️ 중지**: 서비스 중지
   - **🔄 재시작**: 서비스 재시작

## 5️⃣ 설정 변경하기

1. **⚙️ 서버 설정** 클릭
2. 원하는 설정 변경:
   - 서버 IP
   - DHCP 범위
   - 네트워크 인터페이스
   - NFS/TFTP 경로
3. **💾 설정 저장** 버튼 클릭

## 6️⃣ 로그 확인하기

1. **📝 로그 확인** 클릭
2. 드롭다운에서 서비스 선택:
   - dnsmasq
   - nfs-kernel-server
3. **🔄 새로고침** 버튼으로 최신 로그 확인

## 7️⃣ 테마 변경하기

- 사이드바 하단의 **다크 모드** 스위치 클릭
- 즉시 라이트/다크 모드 전환!

## 🎯 주요 단축키

현재 버전에서는 마우스로 모든 작업 가능!

## ⚡ 자주 사용하는 워크플로우

### 새 라즈베리파이 추가
```
클라이언트 관리 → 추가 → 정보 입력 → 추가
→ 자동으로 PXE 설정 생성됨!
```

### 서비스 재시작 (설정 변경 후)
```
서비스 관리 → dnsmasq 재시작 버튼 클릭
→ 즉시 재시작됨!
```

### 문제 발생 시
```
로그 확인 → 해당 서비스 선택 → 에러 메시지 확인
→ 문제 해결!
```

## 🆘 문제 해결

### GUI가 안 열려요!
```bash
# 패키지 설치
pip3 install --user customtkinter psutil netifaces pillow
```

### 서비스 제어가 안 돼요!
```bash
# sudo 권한으로 실행
sudo python3 pxe_gui.py
```

### 화면이 작아요!
```
창 크기 조절 → 최소 1200x700 권장
```
